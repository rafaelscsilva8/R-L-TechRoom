PADTECH S5248-24P

Switch Layer 3 compacto de acesso para redes empresariais e data centers de borda. 24 portas Gigabit (PoE+), 4x 10G SFP+ uplinks, inteligência para VLAN, QoS, roteamento estático e OSPF-lite.

O que é o PADTECH S5248-24P?

Ele é um switch — um equipamento que conecta computadores, câmeras, impressoras e Wi-Fi em uma mesma rede.

  • 24 portas de rede
  • 4 portas SFP+ (mais rápidas)
  • PoE: alimenta câmeras e pontos Wi-Fi
  • Gerenciamento por navegador

Como ligar

  1. Conecte o cabo de energia.
  2. Conecte computadores nas portas 1–24.
  3. Se tiver câmeras/Wi-Fi PoE, conecte nas portas PoE.
  4. Ligue um notebook ao switch para configurar.

Como acessar a configuração

  1. No navegador, digite: http://192.168.10.2
  2. Entre com usuário admin

Configurações básicas (CLI)

1. Configurar IP do switch
enable
configure terminal

interface vlan 1
 description MANAGEMENT
 ip address 192.168.10.2 255.255.255.0
 no shutdown

ip default-gateway 192.168.10.1

write memory
2. Criar VLANs
configure terminal

vlan 10
 name ALUNOS

vlan 20
 name ADMINISTRATIVO
3. Colocar portas dentro das VLANs
interface ethernet 1/0/1
 switchport mode access
 switchport access vlan 10

interface ethernet 1/0/2
 switchport mode access
 switchport access vlan 20

write memory
4. Ativar/Desativar PoE
interface ethernet 1/0/6
 power inline static 15000     ! limita a 15W
! ou
 power inline never            ! desativa PoE
