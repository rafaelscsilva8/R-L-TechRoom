SCMC-4B5L

A "caixa de comunicação de emergência" para redes de fibra óptica. Cria um canal separado só para controle, voz e monitoramento.

1. O que é?

É uma "caixa de comunicação de emergência" para redes de fibra óptica. Imagine que você tem uma rodovia de dados (a fibra óptica principal) e precisa de um "rádio walkie-talkie" para os operadores se comunicarem e gerenciarem essa rodovia, independentemente do tráfego normal. Essa caixa faz exatamente isso: cria um canal separado só para controle, voz e monitoramento, em uma cor de luz específica (1510 nm) que não atrapalha os dados principais.

2. Como funciona?

Funciona como um "tradutor e repetidor de mensagens" para a equipe de rede:

  • Pega mensagens de texto e voz do computador de controle e as transforma em piscadas de luz especiais (1510 nm).
  • Envia essas piscadas por dois lados da fibra (Norte e Sul), como se fosse uma conversa em grupo.
  • Quando recebe piscadas de outras estações, transforma de volta em texto/voz para o operador.
  • Também pode conectar um telefone comum para que técnicos em locais diferentes conversem diretamente pela fibra.
  • É bidirecional e repetidor: pode pegar uma mensagem de um lado e repassar para o outro, estendendo a comunicação.

3. Portas e conectores do dispositivo

Onde conectar as fibras (conectores verdes/azuis):

  • NORTH (Tx1/Rx1): Para a fibra que vai para o Norte.
  • SOUTH (Tx2/Rx2): Para a fibra que vai para o Sul.

Onde conectar acessórios:

  • Tomada de telefone (RJ-11) - para ligar um telefone analógico e conversar.
  • Tomada de linha (RJ-11) - para conectar à rede telefônica externa.
  • Porta de dados (G.703) - para enviar dados lentos de controle.
  • Porta serial - para configuração inicial no computador.

Na parte da frente:

3 luzinhas de status:

  • Verde (POWER): está ligado
  • Vermelha (LOS1): perdeu sinal do lado Norte
  • Vermelha (LOS2): perdeu sinal do lado Sul

4. Características elétricas

  • Voltagem necessária: -48 Volts (recebe do próprio rack automaticamente).
  • Consumo de energia: Muito baixo (cerca de 180 mA, como uma lâmpada LED pequena).
  • Durabilidade: Feito para durar 200.000 horas (mais de 22 anos ligado).

5. Como instalar?

  1. Coloque no rack: Encaixe no slot específico para "Canal de Supervisão".
  2. Conecte as fibras:
    • Fibra do lado Norte nas portas NORTH (Tx1/Rx1)
    • Fibra do lado Sul nas portas SOUTH (Tx2/Rx2)
  3. Empurre totalmente o módulo para dentro do rack (a energia liga sozinha).
  4. Conecte o telefone ou cabos de dados se precisar.

6. Como ligar?

  1. Quando você empurra o módulo totalmente para dentro do rack, ele liga automaticamente.
  2. A luz POWER (verde) acende.
  3. Se as luzes LOS1 e LOS2 (vermelhas) estiverem apagadas, significa que está recebendo sinal dos dois lados.
  4. Se alguma luz vermelha acender, significa que perdeu a comunicação naquele lado.

7. Configurações básicas do dispositivo

  • Praticamente nenhuma! É quase "plug and play".
  • Pode ser necessário conectar um computador via cabo serial na primeira vez para ajustes finos (usando um programa como o TeraTerm).
  • Este modelo específico (SCMC4B5L) NÃO tem a função de "desligamento automático de laser" (ALS) - uma segurança que alguns modelos mais caros têm.

8. Manutenção e boas práticas

CUIDADO AO TIRAR DO RACK (IMPORTANTE):

  1. Use pulseira antiestática.
  2. NÃO pode desligar pelo software.
  3. Para tirar com segurança: Puxe o módulo pela metade para fora do rack (isso corta a energia).
  4. Só então desconecte os cabos de fibra.
  5. Aí pode tirar completamente.

O que ele NÃO faz (neste modelo):

  • Não envia alarmes para o computador central.
  • Não mostra métricas de consumo ou erros remotamente.
  • Você tem que olhar as luzinhas na frente para saber se está tudo bem.

9. Vantagens:

  • Canal de emergência sempre aberto: Permite comunicação mesmo se a rede de dados cair.
  • Voz sobre fibra: Pode ligar um telefone e conversar diretamente com outros técnicos.
  • Longo alcance: Versão "L" é feita para longas distâncias (até dezenas/quedas de quilômetros).
  • Simples e confiável: Poucas peças móveis, dura muitos anos.

10. Limitações:

  • "Cego" para o sistema central: Não manda informações para a sala de controle - você precisa ir olhar no rack.
  • Velocidade baixa: Só para voz e mensagens de texto pequenas (512 kb/s), não para vídeo ou dados pesados.
  • Não serve para equipamentos modernos muito rápidos: Não funciona com transmissores de 40G ou 100G.
  • Sem proteção ALS: Se a fibra for cortada, não desliga automaticamente os lasers potentes (precisa de outro modelo para isso).
